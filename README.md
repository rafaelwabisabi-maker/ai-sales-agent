# AI Sales Agent

An embeddable AI-powered sales chat widget that qualifies leads and books meetings automatically.

## About

AI Sales Agent is a lightweight, configurable chat widget that businesses can embed on their websites. Powered by Claude, it engages visitors in natural conversation, qualifies them against custom criteria, and routes qualified leads to booking flows. Designed for small businesses and consultancies that need 24/7 lead qualification without hiring SDRs.

## Tech Stack

- Node.js / Express
- Anthropic Claude SDK
- SQLite (better-sqlite3)
- Vanilla JS embeddable widget
- Docker / Railway deployment
